# raw/ — Fuentes crudas

Aquí van **tus** documentos originales, tal como los exportas o los escribes.

## Reglas

- **Inmutable.** Claude nunca modifica, renombra ni borra nada aquí. Es la
  fuente de verdad original.
- Cada subcarpeta corresponde a una fuente distinta de información.
- Si una fuente nueva no encaja en las subcarpetas existentes, crea una nueva
  subcarpeta con su propio `README.md`.

## Subcarpetas

| Carpeta            | Qué va aquí                                              |
| ------------------ | -------------------------------------------------------- |
| `claude-exports/`  | Conversaciones exportadas de Claude.                     |
| `chatgpt-exports/` | Conversaciones exportadas de ChatGPT.                    |
| `notion-exports/`  | Páginas y bases de datos exportadas de Notion.           |
| `granola-exports/` | Notas y transcripciones de reuniones de Granola.         |
| `articles/`        | Artículos, papers y lecturas guardadas.                  |
| `notes/`           | Notas sueltas y borradores propios.                      |
