# journal/ — Notas diarias

Diario del second brain (Parte 2).

**Qué va aquí:** una nota por día, normalmente con nombre `YYYY-MM-DD.md`. Sirve
para registrar qué pasó, en qué trabajaste, ideas del día y enlaces a las notas
de `wiki/` que tocaste.

Conéctalas con `[[wikilinks]]` a conceptos, proyectos y personas para que
aparezcan en el graph view de Obsidian.
