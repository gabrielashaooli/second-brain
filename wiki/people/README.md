# wiki/people/

**Dossiers de personas**: quién es, contexto, en qué proyectos aparece, citas y
hechos relevantes recopilados de las fuentes.

Cada nota:
- Empieza con un TLDR de 1–2 frases.
- Cita su(s) fuente(s) de `raw/`.
- Se conecta con otras notas mediante `[[wikilinks]]` (p. ej. a `projects/`).

Al crear o editar una nota, actualiza `../index.md` y añade una línea a
`../log.md`.
