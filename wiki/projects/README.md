# wiki/projects/

Notas **por proyecto**: objetivos, estado, decisiones, hitos y enlaces a las
personas y conceptos implicados.

Cada nota:
- Empieza con un TLDR de 1–2 frases.
- Cita su(s) fuente(s) de `raw/`.
- Se conecta con otras notas mediante `[[wikilinks]]` (p. ej. a `people/`).

Al crear o editar una nota, actualiza `../index.md` y añade una línea a
`../log.md`.
