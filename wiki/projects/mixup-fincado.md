# Mixup / Fincado (trabajo pasado)

**TLDR:** Proyecto de un trabajo anterior (vía StrideMx): sistema para Mixup / iShop Mixup (retail) con backend, web, punto de venta (POS) e impresión de tickets. Backends en NestJS y frontend web en React + Vite.

## Componentes

- **MixupFincado-BE** — backend (NestJS).
- **MixupFincado-Web** — frontend web (React + TypeScript + Vite).
- **MixupPOS-MWBE** — backend del punto de venta / middleware (NestJS).
- **tickets** — servicio que genera e imprime tickets/recibos térmicos de iShop Mixup (ESC/POS, logo iShop Mixup, recibos estilo BBVA), en NestJS.

## Contexto

Trabajo de empleo anterior (org GitHub `StrideMx`), no venture personal. Se guarda como referencia de experiencia y stack usado, no como proyecto activo.
