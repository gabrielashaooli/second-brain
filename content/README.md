# content/ — Pipeline de contenido

Tubería de creación de contenido (Parte 2).

**Qué va aquí:** borradores y piezas en producción (posts, hilos, ensayos,
guiones...) destilados a partir del conocimiento de `wiki/`. Es la salida de
cara al exterior, frente a `wiki/`, que es conocimiento interno.

Sugerencia de flujo: idea → borrador → revisión → publicado. Enlaza cada pieza
con sus fuentes de `wiki/` mediante `[[wikilinks]]`.
